StreamONER

アプリ名: StreamONER

StreamONER とは

StreamONER は、配信者向けのデスクトップアプリケーションです。 OBS で使う配信オーバーレイの表示と操作を助けることが目的です。

アプリの目的と機能

StreamONER は、次のユーザー向け機能を提供します。

  • Discord のボイスチャンネル参加者表示と発話ハイライトを、OBS のブラウザソースに表示する
  • YouTube ライブチャットを、配信中に OBS のブラウザソースへ表示する
  • マイク入力に連動するアバター（口パク）オーバーレイを表示する
  • これらのオーバーレイを配信制作のワークフローに合わせてまとめて使う

対応環境: Windows / macOS 向けデスクトップアプリ（配布インストーラは主に Windows x64）。

Google / YouTube データが必要な理由

StreamONER は、ユーザーが許可した場合に限り、Google の認可を使って YouTube データへ読み取り専用でアクセスします。 利用スコープは https://www.googleapis.com/auth/youtube.readonly のみです。

この権限は、次の目的にだけ使います。

  1. 認可したユーザーの、配信中の YouTube ライブを検出する
  2. ライブチャットを読み取り、StreamONER / OBS 上に表示する

StreamONER は Google のパスワードを取得しません。 認証は Google の同意画面上で行われます。
